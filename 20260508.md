## 1. COOKIES (Memória e Persistência)



### 📖 1. Explicação
No Javascript, as variáveis têm "amnésia": se você apertar F5, todos os dados somem. Os **Cookies** são como pequenos arquivos de texto que o navegador guarda no computador do usuário. Eles sobrevivem ao F5 e até ao fechamento da aba. 

Para gravar um cookie, usamos: `document.cookie = "chave=valor; max-age=tempo_em_segundos"`.
Para ler, verificamos se ele existe com o `includes()`.

Neste módulo, também vamos manipular o CSS via Javascript para alterar o visual da página. O comando `document.body.style.backgroundColor = "black"` muda a cor de fundo, e `document.body.style.color = "white"` muda a cor do texto.

### 💻 2. Exemplo: Acesso Furtivo
No exemplo abaixo, salvamos uma preferência visual. Se o agente clicar em "Modo Escuro", gravamos isso no cookie. Quando a página carrega (`onload`), ela lê o cookie e pinta a tela de preto automaticamente.

```html
<body onload="verificarTema()">
    <h1>Painel de Controle</h1>
    <button onclick="ativarModoEscuro()">Ativar Modo Escuro</button>
    <button onclick="limparRastros()">Limpar Rastros (Sair)</button>

    <script>
        // 1. Função para alterar o visual e salvar no cookie
        function ativarModoEscuro() {
            // Muda o visual
            document.body.style.backgroundColor = "black";
            document.body.style.color = "white";
            
            // Salva o cookie por 1 dia (86400 segundos)
            document.cookie = "tema=escuro; max-age=86400";
        }

        // 2. Função que roda assim que o site abre (onload)
        function verificarTema() {
            // Se o cookie existir e tiver o valor "escuro"
            if (document.cookie.includes("tema=escuro")) {
                document.body.style.backgroundColor = "black";
                document.body.style.color = "white";
            }
        }

        // 3. Função para deletar o cookie (zerando o max-age)
        function limparRastros() {
            document.cookie = "tema=; max-age=0";
            location.reload(); // Recarrega a página
        }
    </script>
</body>
```

## 2. XML (Interceptando Sinais e Feeds)

### 📖 1. Explicação
O XML é um formato clássico de transferência de dados, muito usado em Feeds de Notícias (RSS). Ele parece muito com o HTML, cheio de tags como `<title>` e `<link>`. 

O Javascript lê esse texto e, através do comando `DOMParser`, converte isso em um **Vetor de Itens** (uma lista). Cada notícia fica em uma posição do vetor: `[0]` é a primeira notícia, `[1]` é a segunda, e assim por diante.

### 💻 2. Exemplo: Lendo as Duas Últimas Interceptações
Veja como acessamos as posições `0` e `1` do vetor para preencher links na tela.

```html
<div style="border: 1px solid gray; padding: 10px;">
    <h3>📡 Central de Interceptação:</h3>
    <p>1: <a id="link_zero" href="#" target="_blank">Buscando...</a></p>
    <p>2: <a id="link_um" href="#" target="_blank">Buscando...</a></p>
</div>

<script>
    // Inicia a busca (fetch) no arquivo XML externo
    fetch('https://curso.chr.eti.br/dados/noticias.xml')
        .then(resposta => resposta.text())
        .then(textoXML => {
            // Converte o texto para um formato navegável
            let conversor = new DOMParser();
            let xml = conversor.parseFromString(textoXML, "text/xml");
            
            // Cria o VETOR isolando todas as tags <item> (que são as notícias)
            let vetorNoticias = xml.getElementsByTagName("item");

            // --- LENDO A POSIÇÃO ZERO [0] ---
            let titulo0 = vetorNoticias[0].querySelector("title").textContent;
            let url0 = vetorNoticias[0].querySelector("link").textContent;
            document.getElementById("link_zero").innerText = titulo0;
            document.getElementById("link_zero").href = url0;

            // --- LENDO A POSIÇÃO UM [1] ---
            let titulo1 = vetorNoticias[1].querySelector("title").textContent;
            let url1 = vetorNoticias[1].querySelector("link").textContent;
            document.getElementById("link_um").innerText = titulo1;
            document.getElementById("link_um").href = url1;
        });
</script>
```

## 3. JSON (Rastreamento Governamental em Tempo Real)



### 📖 1. Explicação
O JSON é a evolução do XML. Ele é o formato padrão das APIs modernas (como as do governo) e já entra no Javascript nativamente como um **Vetor de Objetos**. 

Como essas listas costumam ser enormes (ex: todos os estados do Brasil), nós não fazemos manualmente `[0]`, `[1]`, etc. Nós usamos o comando `.forEach()`. Ele cria um "loop" que passa por cada item da lista automaticamente. Junto com ele, usamos o `document.createElement("option")` para construir caixas de seleção `<select>` no HTML sem precisar digitar um por um.

### 💻 2. Exemplo: Rastreador de Estados (IBGE)
Este código busca todos os estados do Brasil e preenche um `<select>`.

```html
<h3>📍 Rastreador Nacional</h3>
<label>Selecione a UF Alvo:</label>
<select id="lista_estados">
    <option>Conectando ao satélite...</option>
</select>

<script>
    // Acessando a API real do IBGE
    fetch('https://servicodados.ibge.gov.br/api/v1/localidades/estados?orderBy=nome')
        .then(resposta => resposta.json()) // Transforma a resposta em JSON
        .then(vetorEstados => {
            let select = document.getElementById("lista_estados");
            select.innerHTML = "<option>Escolha um Estado</option>"; // Limpa o "Conectando..."

            // O forEach varre o vetor. Para cada 'estado', ele executa o bloco:
            vetorEstados.forEach(estado => {
                let opcao = document.createElement("option"); // Cria a tag <option>
                opcao.value = estado.sigla;                   // O valor oculto é a sigla (ex: SP)
                opcao.text = estado.nome;                     // O texto visível é o nome (ex: São Paulo)
                select.add(opcao);                            // Adiciona a opção no Select
            });
        });
</script>
```
