# Parte teórica

## 1. A história do AJAX

A web nasceu em 1991 **síncrona**: clicava num link, o servidor mandava a página inteira, tudo piscava. Cadastro com erro? Tela branca, voltava e você redigitava tudo.

Em 1999 a Microsoft inventou o `XMLHttpRequest` (XHR) — JavaScript falando com servidor **sem recarregar**.

Em 2004/2005, Gmail e Google Maps mostraram pra todo mundo que dava pra fazer site se comportar como aplicativo de desktop.

Em fevereiro de 2005, Jesse James Garrett batizou: **A**synchronous **J**avaScript **a**nd **X**ML.

Em 2006, John Resig lança o **jQuery**. Antes, fazer AJAX dava 25 linhas; com jQuery viraram 3. Em 2010, mais da metade dos sites do mundo usavam jQuery.

Hoje React/Vue/Angular dominam, mas **~70% dos sites ainda têm jQuery**. Aprender jQuery vale porque é o jeito mais didático de entender AJAX e porque você vai pegar manutenção de sistema antigo.

> **AJAX em uma frase:** pedir dados ao servidor em segundo plano e atualizar só uma parte da página.

## 2. Instalando jQuery

É um arquivo só. Coloque antes do `</body>`:

```html
<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
```

Conferir no console (F12):

```javascript
$.fn.jquery    // "3.7.1"   ← se aparecer, jQuery carregou
```

## 3. A função `$()`

`$` é apelido para `jQuery`. Recebe um seletor CSS e devolve uma **coleção**:

```javascript
$('#titulo')        // elemento de id "titulo"
$('p')              // todos os <p>
$('.aviso')         // todos com classe "aviso"
```

A coleção tem métodos. **O método age em todos os elementos** — sem `for`.

```javascript
$('p').css('color', 'blue');   // pinta TODOS os <p> de azul
```

Métodos comuns: `.text()`, `.html()`, `.val()`, `.css(k,v)`, `.addClass(c)`, `.append(x)`, `.remove()`.

Quase todos devolvem a coleção, então dá pra **encadear**:

```javascript
$('#t').text('Novo').css('color', 'red').addClass('grande');
```

## 4. Manipulando o DOM com botões

Vocês já viram `addEventListener` em JS puro. Em jQuery a sintaxe é:

```javascript
$('#btn').on('click', function() {
    // o que fazer quando clicar
});
```

Dentro da função, `$(this)` se refere ao elemento clicado.

Inserção:

| `.append(x)`    | dentro, no fim          |
| `.prepend(x)`   | dentro, no início       |
| `.after(x)`     | depois (fora, irmão)    |
| `.before(x)`    | antes (fora, irmão)     |
| `.remove()`     | apaga                   |
| `.empty()`      | esvazia o conteúdo      |

Efeitos: `.fadeIn(ms)`, `.fadeOut(ms)`, `.slideUp(ms)`, `.slideDown(ms)`.

## 5. AJAX — primeiro contato

```javascript
$.get('url', function(resposta) {
    // este código roda DEPOIS que a resposta chega
});
```

A função no 2º argumento é o **callback** — só roda depois que o servidor responde.

**Atenção:** o código abaixo do `$.get` continua executando **antes** da resposta chegar. Se você precisa do dado, use **dentro** do callback.

API de teste — **JSONPlaceholder** (pública, sem cadastro):

| URL                                                  | Devolve            |
|------------------------------------------------------|--------------------|
| `https://jsonplaceholder.typicode.com/posts/1`       | post #1            |
| `https://jsonplaceholder.typicode.com/users/3`       | usuário #3         |
| `https://jsonplaceholder.typicode.com/users`         | lista de 10        |
