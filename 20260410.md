## 1. Operadores de Comparação e Lógicos
Os operadores são a base da tomada de decisão no código. Eles comparam valores e retornam `true` (verdadeiro) ou `false` (falso).

| Operador | Descrição | Exemplo |
| :--- | :--- | :--- |
| `==` | Igual a (apenas valor) | `x == 8` |
| `===` | Estritamente igual (valor e tipo) | `x === "8"` (falso) |
| `!=` | Diferente de | `x != 5` |
| `>` / `<` | Maior / Menor | `x > 10` |
| `&&` | "E" (ambos verdadeiros) | `idade > 18 && temCnh == true` |
| `\|\|` | "OU" (um deles verdadeiro) | `dia == "Sábado" \|\| dia == "Domingo"` |

## 2. Captura e Conversão de Dados (Casting)
No dia a dia web, pegamos valores de `input`. **Atenção:** Todo dado coletado de um input chega ao JavaScript como uma `string` (texto), mesmo que o usuário digite um número.

* **Por que converter?** Se você tentar somar `"20" + "20"`, o resultado será `"2020"` (concatenação).
* **Como converter:** Use `Number(valor)` ou `parseInt(valor)` para transformar texto em número.

```javascript
// Exemplo real de captura e conversão
const inputIdade = document.getElementById("idade").value;
const idadeNumerica = Number(inputIdade); // Agora podemos fazer cálculos matemáticos
```

## 3. Estruturas Condicionais (`if / else`)
Utilizadas para desviar o fluxo do programa com base em condições. O bloco de código só é executado se a condição for verdadeira.

### Exemplo: Validação de Formulário
```javascript
function validar() {
    let nome = document.getElementById("txtNome").value;
    let idade = Number(document.getElementById("txtIdade").value);

    if (nome === "") {
        alert("O nome deve ser preenchido!");
    } else if (idade < 18) {
        alert("Cadastro permitido apenas para maiores de idade.");
    } else {
        console.log("Dados enviados com sucesso!");
    }
}
```

## 4. Estruturas de Repetição (`for` e `while`)
Usadas para repetir um bloco de código várias vezes de forma automatizada.
* **`for`:** Ideal quando sabemos exatamente quantas vezes o ciclo deve se repetir.
* **`while`:** Ideal quando a repetição depende de uma condição que pode mudar dinamicamente.

### Repetição com `for`: Select Box Dinâmico
```javascript
const select = document.getElementById("anoNascimento");

for (let i = 2024; i >= 1950; i--) {
    let opt = document.createElement("option");
    opt.value = i; // Value numérico
    opt.innerHTML = i;
    select.appendChild(opt);
}
```

### Tabela com `while` (Contagem até 20)
```javascript
let i = 1;
let result = "";

while (i <= 20) {
    result += `<tr><td>${i}</td><td>hello</td></tr>`;
    i++;
}
document.getElementById("corpoTabela").innerHTML = result;
```

## 5. Iteração Moderna com `forEach` e Seletores
Para manipular vários elementos na tela ao mesmo tempo, primeiro precisamos selecioná-los e depois percorrê-los.

### Opções do `querySelectorAll`
O `querySelectorAll` é poderoso porque usa as mesmas regras de seleção do CSS para encontrar elementos na página. Ele devolve uma lista (NodeList) com todos os itens encontrados.

* **Por Tag (Elemento HTML):** `document.querySelectorAll("p")` (Pega todos os parágrafos).
* **Por Classe:** Usa o ponto (`.`). Ex: `document.querySelectorAll(".conteudo")`.
* **Por Atributo:** Usa colchetes (`[]`). Ex: `document.querySelectorAll("input[type='text']")`.
* **Aninhados/Compostos:** `document.querySelectorAll("div .conteudo")` (Elementos com classe "conteudo" dentro de uma div).

### Percorrendo a lista com `forEach`
O `forEach` é a forma mais eficiente de percorrer essa lista. Ele executa uma função para cada item encontrado, sem precisar de um contador numérico.

**Exemplo Prático:** Alterando a cor de todos os parágrafos com a classe `conteudo`.

```html
<p class="conteudo">Primeiro parágrafo do texto.</p>
<p class="conteudo">Segundo parágrafo do texto.</p>
<p class="conteudo">Terceiro parágrafo do texto.</p>
```

```javascript
// 1. Seleciona TODOS os elementos que possuem a classe "conteudo"
const paragrafos = document.querySelectorAll(".conteudo");

// 2. O forEach vai "pegar" cada um desses parágrafos, um por vez
paragrafos.forEach(function(paragrafoAtual) {
    // Para cada parágrafo encontrado, mudamos a cor para azul
    paragrafoAtual.style.color = "blue"; 
});
```
