# Parte teórica

## 1. `$.getJSON` e arrays

Atalho que já faz `JSON.parse` automaticamente:

```javascript
$.getJSON(url, function(dados) {
    // 'dados' já é objeto/array JS
});
```

Quando a URL devolve **array**, o callback recebe array. Por exemplo, `https://jsonplaceholder.typicode.com/users` devolve 10 objetos `{id, name, email, address, company, ...}`.

## 2. Iterando com `$.each`

```javascript
$.each(arrayOuObjeto, function(i, item) {
    console.log(i, item);
});
```

- `i` é o índice (0, 1, 2…)
- `item` é o item atual

Funciona com **arrays e objetos**.

## 3. Criando elementos com `$('<tag>')`

Em vez de concatenar string (perigoso e ilegível):

```javascript
var $bloco = $('<div>').addClass('post');
$bloco.append($('<h3>').text(p.titulo));
$bloco.append($('<p>').text(p.corpo));
$('#mural').append($bloco);
```

`.text(x)` **escapa** HTML — se o dado vier com `<script>`, não interpreta. Mais seguro.

**Convenção:** variáveis que guardam objeto jQuery começam com `$` (`$bloco`, `$item`).

## 4. Estados de UI

Toda interface AJAX passa por **3 estados**:

| Estado          | O que mostrar                                        |
|-----------------|------------------------------------------------------|
| **Carregando**  | "Carregando..." em cinza ou um spinner               |
| **Sucesso**     | os dados                                             |
| **Vazio**       | "Nenhum resultado." (nunca branco)                   |

Se a rede é rápida demais e você quer ver o loading, use:

```javascript
setTimeout(function() {
    $.getJSON(url, callback);
}, 1500);
```

## 5. Busca client-side

Como os dados **já estão na tela**, dá pra filtrar sem nova requisição:

```javascript
$('#busca').on('input', function() {
    var termo = $(this).val().toLowerCase();
    $('.item').each(function() {
        $(this).toggle($(this).text().toLowerCase().indexOf(termo) !== -1);
    });
});
```

- `.on('input', ...)` — dispara a cada tecla.
- `.toggle(boolean)` — mostra se true, esconde se false.
- `.each` (sem `$`) — varre coleção DOM (`$.each` varre array JS — não confunda).

## 6. Encadeando requisições

Às vezes você precisa do post **+** autor. Duas chamadas:

```javascript
$.getJSON('.../posts/1', function(post) {
    $.getJSON('.../users/' + post.userId, function(autor) {
        // aqui você tem post E autor
    });
});
```

Quanto mais aninhamento, pior de ler — na Aula 4 a gente vai aprender a encurtar com `.done()`.
