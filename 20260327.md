
# ⚡ Programação Web: Dando Vida à Página com JavaScript (DOM)

Bem-vindos! Nestas duas aulas práticas, vamos transformar páginas estáticas em aplicações interativas. Vamos aprender a manipular o **DOM** (Document Object Model), que é a forma como o JavaScript enxerga e modifica os elementos do HTML em tempo real.

---

## 📅 Aula 01: A Mágica do CSS via JavaScript

Nesta aula, vamos focar no visual. Você vai aprender como usar o JavaScript para trocar cores, tamanhos e criar animações, respondendo às ações do usuário.

### 📖 Teoria: O DOM e a Manipulação de Estilos
* **O que é o DOM?** É a "árvore" de elementos do seu HTML. O JavaScript consegue "pesgar" qualquer galho dessa árvore para modificá-lo.
* **Selecionando Elementos:** * `document.getElementById('meu-botao')`: Pega um elemento específico.
  * `document.querySelector('.minha-classe')`: Pega elementos como se fosse o CSS.
* **Eventos (`addEventListener`):** O JavaScript fica "escutando". Podemos dizer a ele: *"Quando acontecer um 'click' neste botão, rode esta função"*.
* **Manipulando Estilos:**
  * **Modo Direto:** `elemento.style.backgroundColor = "red";` (Bom para coisas rápidas).
  * **Modo Inteligente (Classes):** `elemento.classList.add('dark-mode')` ou `elemento.classList.toggle('escondido')`. (Melhor prática: deixa o visual no CSS e o JS só liga/desliga a classe).

### 💻 Laboratório Prático 01: Muito Estilo e o Menu "Abre/Fecha"

Vamos criar dois projetos visuais. Crie um arquivo `index.html` e coloque as estruturas abaixo.

**Desafio 1: O Botão do Modo Escuro (Dark Mode)**
Vamos criar um botão que muda a cor do fundo da página inteira.

1. **No seu HTML/CSS (O Corpo e a Roupa):**
   ```html
   <style>
     body { transition: 0.5s; background-color: white; color: black; }
     .dark-mode { background-color: #333; color: white; }
   </style>
   
   <button id="btn-tema">Trocar Tema</button>
   ```

2. **No seu JavaScript (O Cérebro):**
   ```javascript
   // 1. Pega o botão e o corpo (body) da página
   const botaoTema = document.getElementById('btn-tema');
   const corpoDaPagina = document.body;
   
   // 2. Fica escutando o clique
   botaoTema.addEventListener('click', function() {
       // O toggle liga a classe se não tiver, e desliga se tiver!
       corpoDaPagina.classList.toggle('dark-mode');
   });
   ```

**Desafio 2 (PLUS): O Menu Hambúrguer (Abre e Fecha)**
Sabe aquele menu lateral de celular? Vamos fazer um!

1. **HTML e CSS (Estrutura):**
   ```html
   <style>
     #menu-lateral {
       width: 200px; height: 100vh; background: #007bff; color: white;
       position: fixed; left: -200px; /* Escondido fora da tela */
       transition: left 0.3s ease;
     }
     #menu-lateral.aberto { left: 0; /* Traz para dentro da tela */ }
   </style>

   <button id="btn-menu">☰ Menu</button>
   <div id="menu-lateral">
     <ul>
       <li>Home</li>
       <li>Sobre</li>
       <li>Contato</li>
     </ul>
   </div>
   ```

2. **A Mágica no JavaScript:**
   ```javascript
   const btnMenu = document.getElementById('btn-menu');
   const menuLateral = document.getElementById('menu-lateral');

   btnMenu.addEventListener('click', function() {
       // Alterna a classe 'aberto' que empurra o menu para a tela
       menuLateral.classList.toggle('aberto');
   });
   ```
   
# 🎨 Dicionário JavaScript para CSS (`element.style`)

Quando você pega um elemento no JavaScript (ex: `let caixa = document.getElementById('caixa')`), você pode mudar qualquer coisa do visual dele usando o `.style`. 

Aqui estão as propriedades mais usadas no dia a dia para você colar e testar!

### ✍️ 1. Textos e Fontes

| O que você quer mudar | Como fica no CSS | Como escrever no JavaScript | Exemplo de Uso |
| :--- | :--- | :--- | :--- |
| **Cor do texto** | `color` | `style.color` | `caixa.style.color = "red";` |
| **Tamanho da letra** | `font-size` | `style.fontSize` | `caixa.style.fontSize = "24px";` |
| **Tipo da fonte** | `font-family` | `style.fontFamily` | `caixa.style.fontFamily = "Arial, sans-serif";` |
| **Negrito** | `font-weight` | `style.fontWeight` | `caixa.style.fontWeight = "bold";` |
| **Alinhamento** | `text-align` | `style.textAlign` | `caixa.style.textAlign = "center";` |

### 🖌️ 2. Fundo e Cores

| O que você quer mudar | Como fica no CSS | Como escrever no JavaScript | Exemplo de Uso |
| :--- | :--- | :--- | :--- |
| **Cor de fundo** | `background-color` | `style.backgroundColor` | `caixa.style.backgroundColor = "#007bff";` |
| **Imagem de fundo** | `background-image` | `style.backgroundImage` | `caixa.style.backgroundImage = "url('foto.jpg')";` |

### 📏 3. Tamanho, Espaço e Bordas

| O que você quer mudar | Como fica no CSS | Como escrever no JavaScript | Exemplo de Uso |
| :--- | :--- | :--- | :--- |
| **Largura** | `width` | `style.width` | `caixa.style.width = "100%";` |
| **Altura** | `height` | `style.height` | `caixa.style.height = "300px";` |
| **Espaçamento externo** | `margin` | `style.margin` | `caixa.style.margin = "10px 20px";` |
| **Espaçamento interno** | `padding` | `style.padding` | `caixa.style.padding = "15px";` |
| **Borda** | `border` | `style.border` | `caixa.style.border = "2px solid black";` |
| **Arredondar cantos** | `border-radius` | `style.borderRadius` | `caixa.style.borderRadius = "50%";` |

### 👁️ 4. Exibição e Visibilidade

Esses são ótimos para fazer coisas aparecerem e desaparecerem na tela!

| O que você quer mudar | Como fica no CSS | Como escrever no JavaScript | Exemplo de Uso |
| :--- | :--- | :--- | :--- |
| **Esconder/Mostrar** | `display` | `style.display` | `caixa.style.display = "none";` (Esconde) <br> `caixa.style.display = "block";` (Mostra) |
| **Transparência** | `opacity` | `style.opacity` | `caixa.style.opacity = "0.5";` (50% transparente) |
| **Sombra** | `box-shadow` | `style.boxShadow` | `caixa.style.boxShadow = "2px 2px 5px gray";` |

### 💡 Exemplo Completo Juntando Tudo:

Se você quisesse criar um botão de "Alerta" estiloso apenas usando JavaScript, o código seria assim:

```javascript
const meuBotao = document.getElementById('btn-alerta');

// Mudando vários estilos de uma vez
meuBotao.style.backgroundColor = "red";
meuBotao.style.color = "white";
meuBotao.style.fontSize = "20px";
meuBotao.style.borderRadius = "10px";
meuBotao.style.border = "none";
meuBotao.style.padding = "10px 20px";
```
