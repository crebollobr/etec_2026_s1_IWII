# Parte teórica

## 1. O que falta do CRUD

CRUD = **C**reate, **R**ead, **U**pdate, **D**elete.

| Sigla     | Método HTTP | Onde vimos     |
|-----------|-------------|----------------|
| **C**reate | POST       | Aula 3          |
| **R**ead   | GET        | Aulas 1, 2 e 3  |
| **U**pdate | PUT        | **Hoje**        |
| **D**elete | DELETE     | **Hoje**        |

PUT e DELETE **não têm atalho** — é tudo `$.ajax({ method: ..., ... })`.

## 2. PUT — atualizar

```javascript
$.ajax({
    url: 'https://curso.chr.eti.br/ajax/api.php?id=7',
    method: 'PUT',
    data: JSON.stringify({ aluno_id: 'ana123', titulo: 'Comprar leite', feito: 1 }),
    contentType: 'application/json',
    dataType: 'json'
})
.done(function(tarefa) {
    // 'tarefa' = versão atualizada que o servidor devolveu
});
```

**Diferenças em relação ao POST:**

- O id vai **na query string** (`?id=7`), não no corpo.
- O corpo é **JSON** (`JSON.stringify(...)` + `contentType: 'application/json'`).

**Atualização parcial:** a `api.php` aceita só os campos que mudaram:

```javascript
// Marca como feito sem mexer no título
data: JSON.stringify({ aluno_id: ALUNO_ID, feito: 1 })
```

## 3. DELETE — apagar

```javascript
$.ajax({
    url: API + '?id=7&aluno_id=' + ALUNO_ID,
    method: 'DELETE',
    dataType: 'json'
})
.done(function(resp) {
    // resp = { ok: true, id: 7 }
});
```

**Confirme sempre antes de apagar.** Apagar é destrutivo.

```javascript
if (!confirm('Apagar essa tarefa?')) return;
```

## 4. Organizando o código

Até agora ficou tudo numa pilha. Pra apps maiores, separe em **3 camadas**:

```javascript
// === CONFIG ===
var API = '...';
var ALUNO_ID = localStorage.getItem('aluno_id');

// === API (só fala com servidor) ===
function listarTarefas() { return $.getJSON(API + '?aluno_id=' + ALUNO_ID); }
function criarTarefa(t) { return $.post(API, {aluno_id: ALUNO_ID, titulo: t}, null, 'json'); }
function atualizarTarefa(id, dados) {
    return $.ajax({
        url: API + '?id=' + id, method: 'PUT',
        data: JSON.stringify($.extend({aluno_id: ALUNO_ID}, dados)),
        contentType: 'application/json', dataType: 'json'
    });
}
function apagarTarefa(id) {
    return $.ajax({
        url: API + '?id=' + id + '&aluno_id=' + ALUNO_ID,
        method: 'DELETE', dataType: 'json'
    });
}

// === UI (só desenha) ===
function renderizar(tarefas) { ... }

// === LIGAÇÃO (eventos) ===
function carregar() { listarTarefas().done(renderizar); }
$('#form').on('submit', function(e) { ... });
carregar();
```

`$.extend({a:1}, {b:2})` → `{a:1, b:2}` (junta objetos).

## 5. Esperar várias requisições

Quando você dispara várias e quer agir só depois de todas terminarem:

```javascript
var promises = [];
$.each(ids, function(i, id) {
    promises.push(apagarTarefa(id));
});

$.when.apply($, promises).done(function() {
    carregar();   // só roda depois que todas voltaram
});
```

function renderizar(tarefas) { ... }
